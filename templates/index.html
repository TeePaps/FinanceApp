Scoring Methodology

Undervaluation (1.0x weight)

Compares current price to estimated fair value based on EPS.

  • -50% undervalued = +50 pts
  • Fair value = 0 pts
  • +50% overvalued = -50 pts

Dividend Yield (1.5x weight)

Rewards consistent dividend payers.

  • No dividend = -30 pts penalty
  • 1% yield = +5 pts
  • 6%+ yield = +30 pts (max)

Selloff Pressure (0.8x weight)

Identifies stocks that have dropped significantly.

  • % off 52-week high = pts
  • Severe selloff bonus: +15 pts
  • Moderate selloff: +10 pts
  • Recent selloff: +5 pts

# Discovery Questions

## Q1: Is real-time price accuracy during market hours important to you?
**Default if unknown:** Yes (most portfolio tracking apps need current prices for decision-making)

## Q2: Do you want updates to run automatically in the background?
**Default if unknown:** Yes (reduces manual intervention, keeps data fresh)

## Q3: Should the system show clear visibility into what data is stale vs fresh?
**Default if unknown:** Yes (transparency helps users understand when to refresh)

## Q4: Do you need the ability to refresh a single ticker on demand?
**Default if unknown:** Yes (useful for quick checks without full index refresh)

## Q5: Is simplifying the UI (fewer update options) more important than granular control?
**Default if unknown:** Yes (simpler is better for most users, power options can be hidden)
